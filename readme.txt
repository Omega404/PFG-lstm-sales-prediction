# Proyecto LSTM - Predicción de Ventas y Ofertas Personalizadas

Sistema de predicción de demanda de productos usando LSTM y generación de descuentos inteligentes.

## Estructura del proyecto
```
proyecto/
├── src/
│   ├── train/
│   │   └── train_product_lstm.py
│   ├── predict/
│   └── recommendations/
├── data/
├── models/
├── plots/
├── requirements.txt
└── README.md
```

## Instalación
```bash
pip install -r requirements.txt
```

## Uso
```bash
python src/train/train_product_lstm.py
```
